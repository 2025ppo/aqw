- **双层架构**：采用 TypeScript 前端（UI/路由）与 Rust 后端（计算/IO）分离的混合架构，通过 Tauri `invoke` 命令进行进程间通信。
- **核心编排**：前端 `expert-router.ts` 负责意图识别与流水线编排，后端 `lib.rs` 统一注册并分发文件系统、数据库及 AI 检索等原子能力。
- **数据闭环**：结合 SQLite（结构化状态）与本地 JSON（非结构化记忆/索引），实现从用户输入到专家执行、再到交付产物与健康度评估的完整业务闭环。
- **安全隔离**：后端通过 RBAC 权限模型与路径白名单机制，严格管控对敏感文件系统及外部 API 的访问。
