- **配置层叠系统**：`ConfigCascade` 类实现四层优先级（内置默认 < 用户全局 < 项目级 < 运行时覆盖），通过 `deepMerge` 策略合并后端下发的 JSON 配置，并提供单例访问接口。
- **提示词模块化引擎**：`prompt-modules.ts` 定义了基于场景（Scene）和专家角色（Expert）的提示词片段库，支持通过关键词匹配和意图识别动态组装 Prompt，并包含工具 Schema 的动态注入逻辑。
- **历史回溯与推荐**：`prompt-module-history.ts` 负责持久化提示词使用轨迹，利用签名去重和文本相似度算法从历史会话中提取并推荐高频使用的提示词模块。
- **外部服务集成**：`provider-registry.ts` 管理 LLM 提供商元数据，支持从后端加载或本地硬编码回退，并与配置系统联动切换当前活跃模型。
- **辅助分析模块**：`project-health.ts` 和 `data-analysis.ts` 分别封装了项目代码保留率/健康度评估的 API 调用与 HTML 渲染，以及数据分析视图的嵌入式网页控制逻辑。
