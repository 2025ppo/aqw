- **入口层**：`cli-workbench.mjs` 作为核心 E2E 驱动，通过 Node.js `spawnSync` 调用 Python 同步 SQLite 状态，并实现 LLM 工具调用循环。
- **验证层**：`prompt-module-replay.ts` 针对提示词组装逻辑进行单元测试，覆盖意图识别与历史回灌场景。
- **辅助层**：`restore-frontend-test-baseline.mjs` 解析日志文件以重放文件系统操作，用于快速重置测试环境。
- **数据边界**：脚本直接读写 `%APPDATA%` 下的 `projects.json`、`chat_history.db` 等应用持久化数据。
