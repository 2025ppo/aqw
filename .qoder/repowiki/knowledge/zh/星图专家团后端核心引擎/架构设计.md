- **Tauri 桥接架构**：采用 `src-tauri/src/lib.rs` 作为核心逻辑库，通过 `#[tauri::command]` 暴露异步接口给前端，实现 Rust 后端与 Web 前端的进程间通信。
- **全局状态管理**：利用 `AppState` 结构体在应用启动时初始化并注入全局 SQLite 连接池，确保所有子模块（如记忆、Token 统计）共享同一数据持久化层。
- **跨模块协作协议**：通过 `collaboration_engine` 和 `blackboard_engine` 定义标准化的任务分发与上下文同步机制，将专家输出、工具调用结果及用户反馈聚合到共享黑板中，驱动 `pipeline_engine` 进行多步流转。
- **统一安全与配置边界**：由 `rbac` 模块提供基于专家角色的细粒度权限控制，`approval_store` 负责危险命令的沙箱审批，`config` 模块则通过层叠合并策略统一管理 LLM、Shell 及 UI 行为配置。
