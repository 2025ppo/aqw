- 采用纯静态 HTML/CSS/JS 架构，无后端依赖。
- 数据驱动设计：`assets/experts-data.js` 定义全局专家数据对象，`assets/experts-ui.js` 封装 UI 渲染逻辑（如 `renderFeatured`, `bindSearch`）并通过 `window.StarAtlasExpertsUI` 暴露接口。
- 页面职责划分：`index.html` 负责产品营销与核心功能展示，`experts.html` 专注专家角色的检索与详情浏览，`license-*.html` 提供开源协议文本。
- 交互模式：利用原生 DOM API 实现模态框（Modal）、横向拖拽滚动及实时搜索过滤，确保轻量级加载体验。
