# 星图专家团工作台（社区版）

> **业内首创：首个将「AI 专家团协作 + 无限可视化画布 + 智能仓库 Wiki」融为一体的本地桌面级工作台**

## 为什么这是革命性的？

当前市面上的 AI 编程工具（如 Qoder 专家团、CodeBuddy WorkBuddy 等）普遍存在以下痛点：

| 痛点 | 现有方案 | 星图专家团工作台 |
|------|----------|------------------|
| **部署方式** | 云端 SaaS，代码需上传 | **纯本地桌面端**，代码零上传，隐私绝对安全 |
| **协作可视化** | 纯文本对话，无白板 | **无限画布 + 草稿绘制**，专家协作过程可视化 |
| **知识沉淀** | 对话结束即丢失 | **智能仓库 Wiki**，自动生成可迭代的知识库 |
| **架构设计** | 只能生成代码，无法设计 | **可视化架构画布**，拖拽式系统设计与逻辑梳理 |
| **Token 管控** | 无配额管理，费用失控 | **项目级 + 用户级双维度词元配额**，精准控费 |

## 核心首创功能

### 1. 本地优先的 AI 专家团（隐私安全业内最强）

- **完全本地运行**：基于 Tauri 构建的桌面应用，所有代码分析在本地完成，API 密钥本地加密存储
- **多专家智能路由**：架构师、开发工程师、测试专家、安全审计等角色自动分工，根据任务复杂度动态组建专家组
- **密钥池统一管理**：支持 DeepSeek、OpenAI、Anthropic、阿里云、腾讯云等多厂商密钥统一配置与自动切换

### 2. 无限可视化画布（业内首创将画布融入专家团）

- **项目结构可视化**：自动将代码仓库映射为可交互的拓扑图，文件夹、文件关系一目了然
- **草稿白板**：支持手绘、便签、流程图、箭头连线，专家讨论过程全程可视化记录
- **架构设计模式**：拖拽式组件设计，从需求分析到系统架构设计一站式完成

### 3. 智能仓库 Wiki（首创活体知识库）

- **AI 自动生成**：基于代码结构、注释、提交记录自动生成项目文档
- **持续迭代**：代码变更后 Wiki 自动更新，保持文档与代码同步
- **多模式迭代**：支持手动、自迭代、自动迭代三种模式，满足不同场景需求

### 4. 精准词元管控（企业级成本管控）

- **项目级配额**：为每个项目设置日/月/年 Token 上限
- **用户级配额**：个人用户独立统计，多项目共享配额池
- **专家级分析**：按专家维度统计 Token 消耗，优化资源配置

## 适用场景

- **个人开发者**：本地代码分析、架构设计、技术文档编写
- **开源项目维护者**：自动生成项目 Wiki、贡献者指南
- **技术团队**：私有化部署（企业版），团队知识库建设
- **教育机构**：编程教学、代码审查、项目评审

## 技术栈

- **前端**：Vanilla TypeScript + HTML/CSS
- **桌面框架**：Tauri 2.x（Rust 内核，性能极致）
- **后端**：Rust + SQLite（本地持久化）
- **构建工具**：Vite

## 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) 18+
- [Rust](https://www.rust-lang.org/tools/install) 最新稳定版

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run tauri dev
```

### 构建发布版

```bash
npm run tauri build
```
